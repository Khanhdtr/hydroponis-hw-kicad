# circuit-atention-point-blind
Mạch cảnh báo điểm mù
## Yêu cầu thiết kế
-Nguồn vào 24V DC dùng MC24063 => 5V
- IC dùng STM8S0003F3
- Đọc cảnh báo Ultrasonic ( JSN-SR04T)
- Giao tiếp về trung tâm qua đường RS485
